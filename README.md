Sniper Cerebrix: A Hyperadaptive Cognition Interface for Mapping Visual Perception To Neural-State Imagery via Domain-Aligned Interpolation
Overview: This project presents an advanced Brain-Computer Interface (BCI) system designed to decode brain states from visual perception to mental imagery using multimodal EEG and fMRI signals. The system leverages domain adaptation techniques to bridge the gap between perception and imagery, enabling accurate and interpretable brain state classification.
Key Features
🔹 Multimodal Fusion of EEG (temporal) and fMRI (spatial) data
🔹 Domain Adaptation techniques (RTLC, CORAL, MMD)
🔹 Cross-Attention Transformer for feature-level fusion
🔹 Searchlight-based spatial brain mapping for interpretability
🔹 Robust classification of perception vs mental imagery
🔹 Real-time neural signal processing (simulation-based)
🔹 Multilingual semantic decoding (brain → text/speech/imagery)
Datasets Used:
-OSF fMRI Dataset (Perception vs Imagery)
-BOLD5000 Visual Perception Dataset
-BCI Competition IV-2a (EEG Motor Imagery)
Methodology
🔹 Data Preprocessing
EEG signal filtering and normalization
fMRI spatial preprocessing
🔹 Domain Adaptation
Aligning perception and imagery domains using RTLC, CORAL
🔹 Feature Extraction
Deep neural encoders for EEG and fMRI
🔹 Multimodal Fusion
Cross-Attention Transformer to combine modalities
🔹 Classification
Brain state prediction (Perception vs Imagery)
